## Cómo se usa esta lista

- Cualquier pendiente nuevo que surja en cualquier sesión se agrega acá el mismo día, con los
  5 campos completos — no queda solo en `PROGRESS.md`.
- Antes de cerrar cualquier tarea, se recorre esta tabla completa y se reporta si algo
  cambió — aunque la respuesta sea "sigue exactamente igual".
- Cuando un ítem se cierra, no se borra la fila — se cambia el Estado a 🟢 Resuelto y se
  agrega la fecha, para que quede el historial de que existió y se resolvió.
